# Size Guide Excel Template

## How to Create a Size Guide Excel File

Create an Excel file (.xlsx) with the following columns:

| Size | Bust | Waist | Hips | Length |
|------|------|-------|------|--------|
| XS   | 30-32| 24-26 | 34-36| 90     |
| S    | 32-34| 26-28 | 36-38| 92     |
| M    | 36-38| 28-30 | 38-40| 95     |
| L    | 40-42| 32-34 | 42-44| 98     |
| XL   | 44-46| 36-38 | 46-48| 100    |

## Column Descriptions

- **Size**: The size label (e.g., XS, S, M, L, XL)
- **Bust**: Bust measurement (e.g., "32-34" or "86-91 cm")
- **Waist**: Waist measurement (e.g., "26-28" or "66-71 cm")
- **Hips**: Hips measurement (e.g., "36-38" or "91-96 cm")
- **Length**: Length measurement (e.g., "95" or "95 cm")

## Instructions

1. Create a new Excel file
2. Add the column headers exactly as shown above (case-insensitive)
3. Fill in the measurements for each size
4. Save the file as .xlsx format
5. In the admin dashboard, edit a product
6. Click "Import from Excel" in the Size Guide section
7. Select your Excel file
8. The size guide will be imported and saved to the product

## Notes

- All measurements can be in centimeters or inches (specify unit in the dashboard)
- You can leave any field blank if not applicable
- The Excel file must have the headers in the first row
- Only the first sheet in the Excel file will be read
